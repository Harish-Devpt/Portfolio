Personal Portfolio Using React + Vite, This is my personal portfolio website built using **React**, **Vite**, and **TailwindCSS**, deployed on **Vercel**. It showcases my projects, skills, and experiments with frontend design and animations.
-It is Fully responsive layout
- Smooth scroll & page transitions
- Custom GSAP/Framer Motion animations
- Optimized for performance & accessibility

Tech Stack I use is 
- **Frontend:** React (Vite)
- **Styling:** TailwindCSS
- **Animations:** GSAP / Framer Motion
- **Deployment:** Vercel
